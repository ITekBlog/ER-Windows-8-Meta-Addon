== Description ==

ER Windows 8 Meta Addon allows you to add your logo with a customized background as tile to Windows 8 Metro just like any application.

== Installation ==

* Upload "er-windws-8-meta-addon" folder to the "/wp-content/plugins/" directory
* Activate the plugin through the 'Plugins' menu in WordPress
* View Settings > ER Windows 8 Meta for a usage tutorial.
